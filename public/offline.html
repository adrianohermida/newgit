Modo offline ativo

Seguimos operando mesmo sem internet.

Esta instalacao continua acessivel com os arquivos em cache local. Quando a conexao voltar, os dados dinamicos serao sincronizados novamente.

LLM local Use os fluxos locais e a camada offline para continuar trabalhando sem depender da rede.
Base local Obsidian e Supabase local podem assumir como fallback quando os endpoints remotos falharem.
Retomada segura Assim que a internet retornar, recarregue a pagina para buscar a versao mais recente.
